Our Website Features

Feature 1: Naviagte All Pages

Easily move between different pages of the website with user-friendly navigation options, ensuring a seamless browsing experience. Also Highlight the menu item in the navigator corresponding to the current webpage

Feature 2: Highlight Menu Active Page

Our website menu highlight the menu item in the navigator corresponding to the current webpage

Feature 3: Null Fields Check

Our website Order Page checks if the any field of registration form is empty then show alert message and prevent to submit form.

Feature 4: Autofill Billing Address If CheckeBox is Checked

Our website Order Page checks if the "Same as Delivery Address" checkbox is checked and ensures that the billing address is filled automatically. If the delivery address is not completed when this checkbox is checked, it displays an alert message.

Feature 5: Limit Credit Card Digits according to Type

Our website allows users to choose from three types of credit cards: Visa, MasterCard, and American Express. Based on the selected credit card type, it enforces a 16-digit limit for Visa and MasterCard and a 15-digit limit for American Express.
